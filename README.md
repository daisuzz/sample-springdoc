## sample-springdoc

Sample REST API with [Springdoc-openapi](https://springdoc.org/)

## Usage

```shell
$ ./gradlew bootRun
```

| Doc UI     | URL                                           |
|------------|-----------------------------------------------|
| Swagger UI | `http://localhost:8080/swagger-ui/index.html` |
| Redoc      | `http://localhost:8080/redoc/index.html`      |
| RapiDoc    | `http://localhost8080/rapidoc/index.html`     |
